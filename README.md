# HolidayLawyer.ai — Website Version 3

A clean, professional frontend prototype for an AI-assisted holiday and travel claims platform.

## Main improvements

- White interface with restrained blue accents
- Shorter, easier-to-scan homepage
- Broader focus than flight compensation
- Six supported holiday dispute categories
- Working local document generator
- Responsive desktop and mobile design
- Clear AI and legal-service disclosure

## Supported case types

- Flight delays, cancellations and denied boarding
- Lost, delayed or damaged baggage
- Travel insurance claims
- Hotel and accommodation disputes
- Package holiday problems
- Car rental disputes

## Run locally

Open `index.html` in Chrome.

No installation or API key is required for this version.

## GitHub Pages

Upload these files to the root of your repository:

- `index.html`
- `styles.css`
- `app.js`

Then enable GitHub Pages from the repository settings.

## Important brand wording

The design intentionally uses:

- “AI-assisted travel claims”
- “Professional claim document”
- “You approve every document”
- “Not a law firm, court or government authority”

Do not state that a lawyer reviewed or approved the website unless a real qualified lawyer has actually reviewed it and has agreed to be named.

A future human-lawyer option could say:

> Complex cases can be referred to an independent qualified legal professional.

Only add a named “Reviewed by” badge after obtaining genuine professional review and permission.
