# HolidayLawyer.ai — Compact app design

A mobile-first, GitHub Pages-compatible prototype for flight and baggage claims.

## Design direction

The website is now an app-like workspace instead of a long marketing page:

- one focused screen at a time
- fixed bottom navigation
- short cards and labels
- no long feature sections
- white and light-grey interface with restrained blue accents
- Bruce Rogers as the fixed AI Claims Assistant

## Working features

- flight and baggage case wizard
- route coverage check
- possible claim estimate
- readiness score
- evidence checklist
- claim letter generator
- copy and print/PDF
- airline response analyser
- local case saving and JSON export
- privacy and terms pages
